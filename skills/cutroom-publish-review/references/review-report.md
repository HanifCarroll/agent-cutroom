# Publish Review Report

Write `release/publish-review.md` with:

```md
# Publish Review

Status: pass|warn|fail
Target platform: instagram
Final render: renders/final.mp4

## Checks

- Media verification: pass|warn|fail - evidence path
- Preview frames: pass|warn|fail - inspected paths
- Captions: pass|warn|fail - timestamped notes
- Face framing: pass|warn|fail - timestamped notes
- Audio and music: pass|warn|fail - timestamped notes
- B-roll and images: pass|warn|fail - provenance notes
- Social package: pass|warn|fail - package paths

## Required Fixes

- None, or timestamped fix list.

## Warnings

- Any acceptable residual risk.
```

Use `fail` if any required fix remains. Use `warn` only when the video is usable and the residual issue is explicitly acceptable for the requested platform.
